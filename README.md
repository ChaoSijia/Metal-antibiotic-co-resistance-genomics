# Metal-antibiotic-co-resistance-genomics
Scripts for genomic annotation and comparative analysis of metal–antibiotic co-resistance in bacteria from mine tailings and acid mine drainage.

# Soil metatranscriptome ARG analysis

This repository contains scripts used for antibiotic resistance gene (ARG)
annotation and downstream analysis of soil metatranscriptomic data.

## Contents
- `scripts/`: Bash and R scripts for ARG annotation and statistical analysis
- `environment/`: Software versions and conda environment
- `example_data/`: Example input files (no raw sequencing data)

## Software
- BBMap v38.x
- DeepARG v2.0
- R v4.x
